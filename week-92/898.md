# 信息
* 题目: [898.转置矩阵](https://leetcode.cn/problems/transpose-matrix/)
* 标签: 矩阵

# 题解

> [!info]
> 按照数学定义遍历操作即可

```rust
pub fn transpose(matrix: Vec<Vec<i32>>) -> Vec<Vec<i32>> {
    let (m, n) = (matrix.len(), matrix.first().unwrap().len());
    let mut ret = vec![vec![0; m]; n];
    for i in 0..m {
        for j in 0..n {
            ret[j][i] = matrix[i][j];
        }
    }
    return ret;
}
```
