# 信息
* 题目: [888.镜面反射](https://leetcode.cn/problems/mirror-reflection/)
* 标签: 数学

# 题解

> [!info]
> 每反射一次, 在y轴方向最终上升距离为q,
> 如果上升 n 次后, `n*q`为 p 的整数倍, 则到达了顶点
>
> 也就是求最小公倍数

```rust
pub fn mirror_reflection(p: i32, q: i32) -> i32 {
    let d = lcm(p, q);
    if (d / p) % 2 == 0 {
        return 0;
    } else if (d / q) % 2 == 0 {
        return 2;
    }
    return 1;
}

fn gcd(a: i32, b: i32) -> i32 {
    let (m, n) = (a.max(b), a.min(b));
    if n == 0 {
        return m;
    }
    return gcd(n, m % n);
}
fn lcm(a: i32, b: i32) -> i32 {
    a * b / gcd(a, b)
}
```
